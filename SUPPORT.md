# Support

## Where to Ask for Help

- Bug reports: GitHub Issues
- Feature requests: GitHub Issues
- Usage questions: GitHub Discussions if enabled, otherwise GitHub Issues

## Before Opening an Issue

- Read [README.md](README.md) for install, auth, and command examples
- Check [docs/api-coverage.md](docs/api-coverage.md) for product and endpoint coverage notes
- Check [docs/demos.md](docs/demos.md) for demo and repro guidance

## What to Include

- The command you ran
- Whether you used `KAGI_API_TOKEN` or `KAGI_SESSION_TOKEN`
- The exact error output
- Your Rust and OS versions if the problem looks environment-specific

Do not include real tokens, cookies, or private account data in public reports.
